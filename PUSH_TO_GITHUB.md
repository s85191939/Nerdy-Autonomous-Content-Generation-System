# Push to GitHub

Your initial commit is ready locally. To put it on GitHub:

## 1. Create the repository on GitHub

1. Open **https://github.com/new**
2. **Repository name:** `Nerdy-Autonomous-Content-Generation-System`
3. **Owner:** `s85191939` (or your account)
4. Choose **Public** (or Private if you prefer).
5. **Do not** add a README, .gitignore, or license (this repo already has them).
6. Click **Create repository**.

## 2. Push from your machine

In a terminal, from this project folder:

```bash
cd /Users/sanjitrajendiran/Documents/gauntlet-ai/Nerdy
git push -u origin main
```

If GitHub asks for credentials:

- **Username:** your GitHub username  
- **Password:** use a **Personal Access Token** (Settings → Developer settings → Personal access tokens), not your account password.

After this, your code will be at:
**https://github.com/s85191939/Nerdy-Autonomous-Content-Generation-System**
